# Pluto Gym Managment System

##### Requirements
* Node.js >= 8
* MongoDB >= 3.4

##### Installation

**Step 1:**
First step is to install dependencies.
```shell
npm install
```


**Step 2:**
Seed required data information.
```shell
npm run fresh-install
```


**Step 3:**
Final step, start server.
```shell 
npm run serve
```
